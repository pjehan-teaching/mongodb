RAG avec MongoDB

Retrieval-Augmented Generation

Qu'est-ce que le RAG ?

Un LLM possède des connaissances figées à sa date d'entraînement. Le RAG lui permet d'interroger une base de connaissances à jour avant de répondre.

  1. Retrieval — recherche des documents les plus pertinents par similarité vectorielle
  2. Augmented — injection de ces documents dans le prompt comme contexte
  3. Generation — le LLM génère une réponse ancrée dans ce contexte

Cas d'usage : répondre à des questions sur les sujets E7 BTS SIO sans que le modèle ne les ait jamais lus.

Prérequis — Ollama

Ollama fait tourner des LLM en local. Installation :

# Linux / macOS
curl -fsSL https://ollama.com/install.sh | sh

# Docker
docker run -d -p 11434:11434 --name ollama ollama/ollama

Téléchargement des modèles :

ollama pull nomic-embed-text   # embeddings 768 dimensions (~270 Mo)
ollama pull llama3             # modèle de chat (~4,7 Go)

ollama list                    # vérification

L'API REST d'Ollama écoute sur http://localhost:11434 — c'est cette URL que le projet utilise directement via fetch().

03_insert.ts

import { readdirSync } from 'fs';
import { join } from 'path';
import { RecursiveCharacterTextSplitter } from '@langchain/textsplitters';
import { Chunk, generateEmbedding, getCollection, parsePdf } from './utils.js';

const size = Number(process.env.CHUNK_SIZE) || 500;
const overlap = Number(process.env.CHUNK_OVERLAP) || 50;
const pdfDir = process.env.PDF_DIR ?? './pdfs';

const splitter = new RecursiveCharacterTextSplitter({ chunkSize: size, chunkOverlap: overlap });
const { client, collection } = await getCollection<Chunk>(process.env.COLLECTION_NAME!);

try {
  await collection.deleteMany({});
  const files = readdirSync(pdfDir).filter((f: string) => f.endsWith('.pdf'));

  for (const file of files) {
    const { text } = await parsePdf(join(pdfDir, file));
    const texts    = await splitter.splitText(text);
    console.log(`\n${file} → ${texts.length} chunks`);

    const chunks: Chunk[] = [];
    for (let i = 0; i < texts.length; i++) {
      chunks.push({
        source:     file,
        chunkIndex: i,
        contenu:    texts[i],
        embedding:  await generateEmbedding(texts[i]),
      });
      process.stdout.write(`\r  ⚙  ${i + 1} / ${texts.length} embeddings`);
    }
    await collection.insertMany(chunks);
    console.log(`\n  ✓ ${chunks.length} chunks insérés`);
  }
} finally {
  await client.close();
}

    name: 'vector_index',
    type: 'vectorSearch',
    definition: {
      fields: [
        {
          type: 'vector',
          path: 'embedding',
          numDimensions: 768, // nomic-embed-text produit 768 dimensions
          similarity: 'cosine',
        },
        {
          type: 'filter',
          path: 'source', // filtre optionnel par fichier PDF
        },
      ],
    },
  });

  await client.close();
}

export async function chat(question: string, context: string): Promise<void> {
  const system =
    'Tu es un assistant expert. Réponds uniquement en te basant sur le contexte fourni.\n\n' +
    `Contexte :\n${context}`;

  const res = await fetch(`${process.env.OLLAMA_BASE_URL}/api/chat`, {
    method: 'POST',
    body: JSON.stringify({
      model: process.env.CHAT_MODEL,
      stream: true,
      messages: [
        { role: 'system', content: system },
        { role: 'user',   content: question },
      ],
    }),
  });

  // Lecture du flux NDJSON (une ligne JSON par token)
  const reader = res.body!.getReader();
  const decoder = new TextDecoder();
  let buffer = '';

  while (true) {
    const { done, value } = await reader.read();
    if (done) break;
    buffer += decoder.decode(value, { stream: true });
    const lines = buffer.split('\n');
    buffer = lines.pop()!; // fragment incomplet
    for (const line of lines) {
      if (!line.trim()) continue;
      const { message } = JSON.parse(line);
      if (message?.content) process.stdout.write(message.content);
    }
  }
  console.log();
}

05_rag.ts

import { Chunk, chat, generateEmbedding, getCollection } from './utils.js';

const query = process.argv[2];
if (!query) {
  console.error('Usage: npm run 05 -- "<question>"');
  process.exit(1);
}

const { client, collection } = await getCollection<Chunk>(process.env.COLLECTION_NAME!);

try {
  const topK = Number(process.env.TOP_K) || 5;
  const queryEmbedding = await generateEmbedding(query);

  const chunks = await collection.aggregate<Chunk>([
    {
      $vectorSearch: {
        index:         'vector_index',
        path:          'embedding',
        queryVector:   queryEmbedding,
        numCandidates: topK * 10,
        limit:         topK,
      },
    },
    { $project: { _id: 0, contenu: 1, source: 1, chunkIndex: 1 } },
  ]).toArray();

  // Affichage des sources utilisées
  console.log('\nSources :\n');
  chunks.forEach((c, i) =>
    console.log(`  ${i + 1}. ${c.source}  (chunk #${c.chunkIndex})`)
  );

  // Construction du contexte et appel LLM
  const context = chunks.map(c => c.contenu).join('\n\n---\n\n');
  console.log('\nRéponse :\n');
  await chat(query, context);
} finally {
  await client.close();
}
