Sécurité

Activer l'authentification

Pour activer le système d'authentification, vous devez commencer par créer un nouvel utilisateur :

> use admin
> db.createUser({ user: "admin", pwd: "1234", roles: [{ role: "userAdminAnyDatabase", db: "admin" }, { role: "readWriteAnyDatabase", db: "admin" }] })

Il faut ensuite redémarrer le serveur mongod en activant l'authentification :

mongod --auth
MongoDB Built-In Roles

S'authentifier

Vous pouvez vous connecter au mongo shell en vous authentifiant :

mongo -u "admin" -p "1234" --authenticationDatabase "admin"

Vous pouvez également vous connecter dans un premier temps puis vous authentifier ensuite :

mongo
> use admin
> db.auth("admin", "1234")

Modifier un utilisateur
