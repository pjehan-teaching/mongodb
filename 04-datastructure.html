Structure des données

Databases

  • MongoDB contient une liste de base de données (databases)
  • Chaque base de données contient une liste de collections
  • Chaque collection contient une liste de documents

Format JSON

  • Les champs sont séparés par des virgules

Format BSON (Binary JSON)

Etend le format JSON en apportant de nouveaux types de champs et de meilleurs performances.

  • Lightweight (peu d'espace disque)
  • Traversable (parcourir, écrire, modifier, supprimer...)
  • Efficient (encodage / décodage très performant)
