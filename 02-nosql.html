NoSQL

Définition

En informatique et en bases de données, NoSQL désigne une famille de systèmes de gestion de base de données (SGBD) qui s'écarte du paradigme classique des bases relationnelles.
Wikipedia
  • Not only SQL
  • Pas de tables
  • Pas de relations

Solutions

Type Solutions
Key-value Redis
Document MongoDB, CouchBase
Column DynamoDB, Bigtable, Cassandra
Tuple Apache River
Object Objectivity/DB, Perst
Graph Neo4j, ArangoDB, OrientDB

Quand utiliser une base de données NoSQL ?

  • Gérer une quantité très importante de données
  • Gérer des données non structurées
  • Respecter les principes de développement modernes de la méthode Agile
  • Développer des microservices plutôt qu'une application monolithique
