Qu'est-ce que MongoDB ?

Ses avantages...

  • Gratuit et Open Source
  • Ecrit en C++
  • Stockage des données sous forme de Documents similaires au format JSON
  • Les données sont flexibles et la structure des données peut varier entre les documents mais également dans le temps
  • Les Documents correspondent aux informations utilisées dans les applications ce qui permet de simplifier leur utilisation
  • Scalable avec la possibilité de répartir les données sur plusieurs machines (horizontal scaling)
  • Requêtes avec variables, indexation, aggrégation de données...

SQL to MongoDB

... et ses inconvénients

  • Respecte le principe ACID (Atomic, Consistent, Isolated, Durable) mais uniquement au niveau d'un Document ==> V4
  • Pas de transaction (il est cependant possible de s'en approcher en utilisant le système de Two Phase Commits)
  • Pas de Foreign Keys et donc d'intégrité référentielle
