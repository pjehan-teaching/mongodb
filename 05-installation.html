Installation

Page d'installation

Architecture

MAC OS

mkdir -p /data/db
chmod -R 777 /data
cd ~/Downloads/
tar xvf mongodb-osx.tgz
cp mongodb-osx/bin/* /usr/local/bin/
which mongod
mongod

En cas de problème de droits pour la création du dossier /data/db :

cd ~
mkdir -p data/db
chmod -R 777 data
mongod --dbpath=~/data/db

Le serveur démarre sur le port par défaut : 27017

Windows

  1. Télécharger et installer le fichier .msi
  2. Les fichiers sont installés dans C:\Program Files\MongoDB\Server\[VERSION_MONGODB]\bin\
  3. Ajouter ce répertoire dans le PATH (modifiable depuis les variables d'environnement)
md \data
md \data\db
mongod

Le serveur démarre sur le port par défaut : 27017
